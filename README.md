# Queue implementation
Implementing a queue data structure based on:

1-Arrays
2-Linked lists

## Compiling
First compile the program using [make](https://www3.ntu.edu.sg/home/ehchua/programming/cpp/gcc_make.html)
```bash
make all
```
then run the program
```bash
./queue
```
